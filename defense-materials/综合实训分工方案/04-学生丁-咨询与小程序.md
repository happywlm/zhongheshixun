# 学生丁 — 咨询问答 + 小程序（交互亮点 + 移动端）

> **综合实训角色**：学生丁  
> **业务域**：咨询问答 + 小程序  
> **核心叙事**：「学员提问题 → 机器先答 → 答不了转人工 → 人工 <1min 响应 → 小程序怎么用」

---

## 一、业务定位

你是整组**对外展示效果最好**的角色。你的模块是：

1. **交互亮点** — ⭐ 关键词匹配 + SLA 超时告警（docx 3.5 强制）
2. **移动端闭环** — ⭐ 小程序全 10 页（对外演示震撼）
3. **响应式保障** — ⭐ 应答时间 < 1 分钟（SLA 强制）

回答三个问题：

1. **学员在咨询问什么问题？** — 在线咨询（小程序 + web-student）
2. **系统怎么回答？** — ⭐ 关键词提取 + 知识库 LIKE 命中
3. **答不了怎么办？** — ⭐ 转人工 + 超时标红（SLA 保障）

前置依赖：
- **学生甲** 提供登录 token 和 userId（咨询需要用户身份）
- **学生乙** 的课程、**学生丙** 的试题可能被咨询引用

### 4.1 ⭐ 智能问答接口（ConsultApiController，9899）

**请求**：

```json
  "data": {
    "recordId": 25,

    "matchScore": 0.85
  }
```

**⭐ 关键代码**：

```java
// ConsultServiceImpl.ask()
public ConsultResult ask(String question, Long userId) {
    // 1. 提取关键词
    List<String> keywords = extractKeywords(question);
    // "高血压怎么用药" → ["高血压", "用药"]
    
    // 2. 知识库 LIKE 命中
    List<KnowledgeBase> hits = knowledgeBaseMapper.searchByKeywords(keywords);
    
    if (!hits.isEmpty()) {
        // 命中：返回预设答案，is_auto=1
        KnowledgeBase bestMatch = hits.get(0);
        // 增加点击热度
        knowledgeBaseMapper.incrementClick(bestMatch.getId());
        
        // 写 consult_record
        ConsultRecord record = new ConsultRecord();
        record.setUserId(userId);
        record.setQuestion(question);
        record.setAnswer(bestMatch.getAnswer());
        record.setIsAuto(1);
        record.setStatus(1);
        consultRecordMapper.insert(record);
        
        return new ConsultResult(record.getId(), true, bestMatch.getAnswer());
    }
    
    // 3. 无匹配 → 转人工
    ConsultRecord record = new ConsultRecord();
    record.setUserId(userId);
    record.setQuestion(question);
    record.setAnswer("正在为您转接人工客服...");
    record.setIsAuto(0);
    record.setStatus(0);  // 待回复
    record.setTransferTime(new Date());  // ⭐ 记录转人工时间
    record.setSlaSeconds(60);  // ⭐ SLA 60 秒（演示用）
    consultRecordMapper.insert(record);
    
    return new ConsultResult(record.getId(), false, "正在为您转接人工客服...");
}

// 关键词提取（中文按字符/空格切）
private List<String> extractKeywords(String question) {
    // 简单实现：中文按字符切、空格分、去掉停用词
    String cleaned = question.replaceAll("[，。！？\\s]+", " ").trim();
    return Arrays.stream(cleaned.split("\\s+"))
        .filter(s -> s.length() > 1)  // 过滤单字
        .collect(Collectors.toList());
}

// SQL 关键词匹配
// <select id="searchByKeywords">
//   SELECT * FROM knowledge_base

//   <foreach collection="keywords" item="kw">
//     OR keywords LIKE CONCAT('%', #{kw}, '%')
//   </foreach>
//   ORDER BY click_count DESC
// </select>
```

### 4.2 ⭐ SLA 超时告警机制（ConsultController，9898）⭐ 学习重点

**SLA 告警落地方案**：

```
1. consult_record 三字段记录：
   - transfer_time：转人工时间
   - sla_seconds：SLA 秒数（演示设为 60）
   - sla_exceeded：0=未超时，1=超时
   
2. 管理员后台每 10 秒 Ajax 查一次待队列：
   SELECT * FROM consult_record 
   WHERE is_auto = 0 AND status = 0
     AND TIMESTAMPDIFF(SECOND, transfer_time, NOW()) > sla_seconds;
   
3. 返回的记录 sla_exceeded=1 → 前端红色标红
```

**前端代码**（consult/list.vue）：

```vue
<template>
  <el-table :data="consultList" @cell-style="cellStyle">
    <el-table-column prop="question" label="问题" />
    <el-table-column prop="replyTime" label="回复时间" />
    <el-table-column prop="transferTime" label="转人工时间" />
    <el-table-column prop="sla" label="SLA">
      <template #default="{ row }">
        <el-tag :type="row.slaExceeded ? 'danger' : 'success'">
          {{ row.slaExceeded ? '已超时' : '未超时' }}
        </el-tag>
      </template>
    </el-table-column>
  </el-table>
  <!-- 每 10 秒轮询 -->
  <script setup>
    onMounted(() => {
      fetchConsultList();
      const timer = setInterval(fetchConsultList, 10_000);
      onUnmounted(() => clearInterval(timer));
    });
  </script>
</template>
```

### 4.3 人工回复接口（ConsultController，9898）

**请求**：

```http
Content-Type: application/json

{
  "recordId": 25,
```

**关键代码**：

```java
@PostMapping("/reply")
public Result reply(@RequestBody ReplyDTO dto, HttpServletRequest request) {
    Long adminId = (Long) request.getAttribute("userId");
    
    ConsultRecord record = consultRecordMapper.selectById(dto.getRecordId());
    record.setAnswer(dto.getAnswer());
    record.setIsAuto(0);
    record.setStatus(1);  // 已回复
    record.setReplyTime(new Date());  // ⭐ 记录回复时间
    record.setAdminId(adminId);
    
    // sla_exceeded 计算
    long duration = (record.getReplyTime().getTime() 
                   - record.getTransferTime().getTime()) / 1000;
    record.setSlaExceeded(duration > record.getSlaSeconds() ? 1 : 0);
    
    consultRecordMapper.updateById(record);
    return Result.success();
}
```

### 4.5 小程序登录（WxAuthController，9899）

**请求**（前端 `wx.login` 取 code 后调用，**无需账号密码**）：

```http
POST /api/wx/login
Content-Type: application/json

{
  "code": "0a3xPP000xxx...",
  "nickName": "学员",
  "avatar": ""
}
```

**响应**：

```json
{
  "code": 200,
  "data": {
    "token": "eyJhbGciOiJIUzI1NiJ9...",
    "userInfo": { "id": 20, "username": "wx_4b76c35b", "realName": "微信用户", "role": "STUDENT" }
  }
}
```

> **实现说明（毕设简化版）**：后端 `WxAuthController.wxLogin` 用 `code` 模拟生成 openid（`UUID.nameUUIDFromBytes`），按 `wx_<openid前8位>` 查找/创建学员用户（角色 `STUDENT`，默认密码 `123456` BCrypt 哈希），签发 JWT。
> **生产做法**：后端用 `code` + `appid` + `secret` 调微信 `jscode2session` 接口换真实 openid，其余流程相同。
> ⚠️ **真机联调**：`code` 有效期仅 5 分钟且一次性；手机必须连 WiFi 能访问电脑 9899（详见 `00-项目启动与端口速查.md` 步骤 6），否则报 `ERR_CONNECTION_REFUSED`。

```javascript
// consult/index.js
Page({
  data: {
    messages: [],
    input: '',
    isTyping: false
  },
  
  onSend() {
    const question = this.data.input;
    if (!question.trim()) return;
    
    // 添加学员消息到列表
    this.addMessage('user', question);
    
    // 显示打字机效果
    this.setData({ isTyping: true, input: '' });
    
    // 模拟打字机 1.5 秒
    setTimeout(() => {
      // 调 API（baseURL 已在 api/config.js 动态配置）
      const { baseURL } = require('../../api/config');
      wx.request({
        url: baseURL + '/consult/ask',
        method: 'POST',
        header: { Authorization: 'Bearer ' + wx.getStorageSync('token') },
        data: { question },
        success: (res) => {
          const answer = res.data.data.answer;
          this.addMessage('system', answer);
          this.setData({ isTyping: false });
        }
      });
    }, 1500);
  },
  
  addMessage(role, content) {
    const messages = this.data.messages;
    messages.push({
      id: Date.now(),
      role,
      content,
      time: this.formatTime()
    });
    this.setData({ messages });
    // 自动滚到底部
    wx.pageScrollTo({ scrollTop: 99999 });
  }
});
```

**请求封装 + 微信一键登录**：

```javascript
// api/config.js（baseURL 动态：模拟器走 localhost，真机走电脑局域网 IP）
module.exports = {
  baseURL: (() => {
    let host = '192.168.70.227'  // 电脑局域网 IP，换 WiFi 后需同步修改
    try { if (wx.getSystemInfoSync().platform === 'devtools') host = 'localhost' } catch (e) {}
    return `http://${host}:9899/api`
  })()
}

// api/request.js（统一请求封装，自动注入 token）
const { baseURL } = require('./config')
function request(url, method = 'GET', data = {}) {
  return new Promise((resolve, reject) => {
    wx.request({
      url: baseURL + url,
      method, data,
      header: { Authorization: 'Bearer ' + (wx.getStorageSync('token') || '') },
      success: (res) => resolve(res.data.data),
      fail: reject
    })
  })
}

// pages/login/login.js（微信一键登录，无需账号密码）
wx.login({
  success: (res) => {
    request('/wx/login', 'POST', { code: res.code, nickName: '学员', avatar: '' })
      .then((result) => {
        wx.setStorageSync('token', result.token)
        wx.setStorageSync('userInfo', result.userInfo)
        wx.reLaunch({ url: '/pages/index/index' })
      })
  }
})
```

## 九、常见 Bug 与解决

| 现象 | 原因 | 解决 |
|------|------|------|
| 关键词没命中 | keywords 字段没写或分隔符错 | 检查是逗号不是中文逗号 |
| 命中了但返回错答案 | keywords 归并结果排序错 | 看 click_count DESC 排序 |
| 转人工没写 DB | ask() 漏 insert 待队列 | 看 ConsultServiceImpl |
| 标红没生效 | sla_exceeded 没算 | 看 admin/consult/list.vue calloutFlag |
| 回复后 status 没变 | 后端漏 update status=1 | 看 reply() 流程 |
| 小程序 401 | token 没带或过期 | 看 wx.getStorageSync('token') |
| 小程序网络失败 | baseURL 写错 | 看 app.js globalData.apiBase |
| 小程序视频播不了 | 视频 URL 是 localhost 非 HTTPS | 开发工具勾选"不校验合法域名" |
| PDF 下载失败 | file_url 路径错 | 看 resource_file.file_url |
| 打字机没效果 | isTyping 状态没 toggle | 看 setData 顺序 |
